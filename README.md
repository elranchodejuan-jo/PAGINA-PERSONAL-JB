# Juan Bajaña | Landing Personal

Landing page personal profesional para Juan Bajaña, estudiante de Medicina Veterinaria y creador de proyectos digitales aplicados a educación veterinaria, clínicas y granjas.

## Estructura

- `index.html`: contenido principal y secciones de la landing.
- `css/styles.css`: estilos, variables CSS, responsive design y animaciones.
- `js/main.js`: menú móvil, estado del header, scroll activo y revelado de secciones.
- `assets/juan-bajana-perro.jpeg`: foto principal usada en el hero.

## Cómo abrir

Abre `index.html` directamente en el navegador. También puedes publicarlo en GitHub Pages, Vercel o Netlify porque no requiere backend ni build step.

## Pendientes editables

- Reemplazar los placeholders de imagen por fotos reales o capturas de proyectos.
- Cambiar los `href="#"` de Suite Vet, Cartilla Digital, CATTLE, GitHub, LinkedIn y redes por enlaces reales.
- Agregar un favicon real cuando esté disponible.
